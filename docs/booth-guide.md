## 問 PM 的通用心法

> 1. 看 demo 別只看炫，問「**失敗怎麼辦**」最有料。
> 2. 如果 PM 只能回行銷話術、給不出實測數字 → 標「待查」，不用在現場下結論。
> 3. 大眾日 PM 已經累了 — 問題精簡，一次問一兩個就好。

## 三大技術趨勢（今年值得帶著看的視角）

1. **AI 機櫃電力鏈重構** — 電源廠都在講 rack 級電力架構（Power Shelf、BBU、800V DC）。看點：Vertiv / 光寶 / 肯微。
2. **液冷比的是「可維運性」** — 不再只是能不能散熱，而是壞了怎麼修。看點：雙鴻 / 勤誠 / 神雲。
3. **機器人 demo 要拆著看** — demo 成功 ≠ 量產可用。看點：立端 / 所羅門 / 研華。問「推論跑哪、失敗怎麼復原」。
