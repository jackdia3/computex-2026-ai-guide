## 入場方式

| | 業界人士登錄 | 一般民眾購票 |
|---|---|---|
| 費用 | **免費** | NT$200（優待票 $100） |
| 可入場日 | 6/2 – 6/5 全展期 | **僅 6/5** |
| 辦理方式 | 線上預先登錄已截止 → **現場登錄**（報到櫃台，6/1 起） | [TaiTIX 線上購票](https://taitix.com.tw/event/detail/100003/computex26) 或現場購票 |
| 需要什麼 | 名片 ×2（科技 / 相關產業工作即可） | 身分證件（滿 18 歲） |

### ⚠️ 三個關鍵提醒

1. **有名片就走業界登錄** — 免費且不受民眾票限制。
2. **民眾票的展館範圍有爭議** — 部分媒體報導民眾票只能進世貿一館，官方未明確寫。要去南港的人行前確認，或直接辦業界登錄。
3. **未滿 18 歲不能入場**；現場售票只到 **15:00**。
