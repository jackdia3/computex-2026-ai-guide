## 入場方式

| | 專業人士／投資者登錄 | 一般民眾購票 |
|---|---|---|
| 費用 | **免費** | NT$200（優待票 $100） |
| 可入場日 | 6/2 – 6/5 全展期 | **僅 6/5** |
| 辦理方式 | 線上預先登錄已截止 → 展期間至**現場報到櫃台登錄** | [TaiTIX 線上購票](https://taitix.com.tw/event/detail/100003/computex26) 或現場購票 |
| 需要什麼 | 名片＋工作相關證明，**資格以現場審核為準** | 身分證件（滿 18 歲） |

### ⚠️ 三個關鍵提醒

1. **在科技／相關產業工作的人，建議走專業人士登錄** — 免費且全展期可入場；帶名片，資格以現場審核為準。
2. **民眾票於 6/5 可參觀**：南港 1、2 館至 15:30、世貿一館至 16:30（官方與售票頁均列出兩地時間）。若現場票務或動線臨時調整，以官方 APP 與入口公告為準。
3. **未滿 18 歲不能入場**；現場售票只到 **15:00**。
